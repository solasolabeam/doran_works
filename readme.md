# DoranWorks

### 프로젝트 기획 및 소개
> **기획 목적**

메신저, 쪽지, 전자결재 시스템 등 업무에 필요한 기능을 하나로 통합한 그룹웨어입니다.

일상생활에서 접하는 메신저와 쪽지 기능, 업무와 밀접한 전자결재 시스템을 개발하며 사내 시스템에 대해 더욱 잘 이해하기 위해 도란웍스를 개발하게 되었습니다.
